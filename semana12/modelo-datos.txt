======================================================================
EXPLICACIÓN DEL PROPÓSITO DE CADA ENTIDAD
======================================================================

1. Entidad "Causa":
Sirve para almacenar la información de las diferentes iniciativas o 
peticiones creadas por la comunidad. Centraliza los datos del problema 
(título, descripción detallada y fecha de creación) para que puedan ser 
visualizados por otros usuarios y sirvan de base para generar el 
documento formal final.

2. Entidad "Apoyo":
Sirve para registrar la firma de respaldo de cada ciudadano a una causa 
específica. Almacena la relación mediante el campo 'causa_id' 
(para saber a qué petición apoya), el nombre del firmante y el momento 
exacto en el que otorgó su apoyo, permitiendo al sistema contar las 
firmas acumuladas de manera automatizada.
