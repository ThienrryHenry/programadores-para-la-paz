Al enviar una petición POST a la URL de mi Codespace:
https://expert-space-doodle-699rvrg5w4gx2rqvx-3000.app.github.dev/registro

Con el JSON de María, el servidor devuelve un objeto en formato JSON con un código 
de estado exitoso (200 OK).

La respuesta exacta del servidor es:
{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Esto ocurre porque el middleware express.json() procesa el cuerpo de la petición en la 
nube de GitHub, permitiendo que 'req.body.nombre' y 'req.body.mensaje' capturen la 
información y la devuelvan estructurada en la respuesta.
